## React技术栈
### 目的

1. 快速入手、学习、深入react技术栈
2. 包括react概述，事件系统，css模块化，组件通信，高阶组件，渲染劫持，性能优化，动画，测试，react源码，flux,redux,服务端渲染，可视化等内容

### 使用方法

1. npm i
2. npm start
3. 浏览器输入localhost:8000

### 文档

文档均在doc目录下。每章单独markdown文件。

    本项目目标快速入手、学习、深入react技术栈，其他功能请参考。
